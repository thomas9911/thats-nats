# ThatsNats

## Running nats

```sh
docker-compose up -d
```

## Running services

### Elixir

`iex -S mix` from the root


### Rust

`cargo run` from the root


### Go

`go run main.go` from services/thats-nats-go

or

`go run .\services\thats-nats-go\` from the root


### Starting all

```sh
. run.sh
```

## Testing

```sh
. test.sh
```
